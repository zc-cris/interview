#### 2. duplicate包：利用set的特性快速给list集合去重

#### 6. polymorphism包：java中的多态练习

#### 7. staticorder包：java中对象的生成顺序

#### 8. string包：java中的String类型（常量池，intern方法入池，以及+符号的两种形式）
